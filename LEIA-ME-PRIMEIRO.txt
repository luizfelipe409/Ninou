NINOU v84.1.1 — BASE UNIVERSAL INTEGRADA

A web, o iPhone e o Android agora são gerados pelo mesmo aplicativo Expo/React Native.
Não existe mais uma implementação web separada para perfil, rotina, órbita ou menus.

INSTALAÇÃO MOBILE
cd mobile
npm ci
npm run typecheck
npm run start:dev-client -- --clear

VALIDAÇÃO E BUILD WEB
cd mobile
npm ci
npm run export:web

O deploy da Vercel usa mobile/dist e o mesmo Firebase/Firestore das plataformas nativas.
A pasta node_modules não acompanha o ZIP e é recriada com npm ci.
