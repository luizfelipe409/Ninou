# Ninou v75.9 - Admin com contexto limpo

Versão baseada na v75.8, com correção de troca de contas e melhorias de espaçamento no painel Administração do App.

Principais ajustes:

- Ao entrar com o admin, o app não reutiliza mais foto/perfil/rotina da conta que estava logada antes.
- Ao sair da conta, a visualização volta para modo visitante sem dados privados.
- Cache local passou a ser separado por e-mail, evitando vazamento visual entre contas no mesmo iPhone.
- A ativação do admin não envia perfil/foto/rotina antiga para a família principal se os dados não pertencem ao admin.
- Painel Administração do App com espaçamentos revisados para iPhone.
