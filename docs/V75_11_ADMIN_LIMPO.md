# Ninou v75.11 — Admin limpo e profissional

Esta versão reorganiza a área de Administração do Ninou para mostrar apenas informações pertinentes ao administrador do app.

## O que mudou

- O painel Admin agora é focado em controle de acesso.
- Admin vê somente: resumo, criar convite, convites pendentes e usuários autorizados.
- Removido o visual confuso de “admin aguardando convite”.
- Campo de aceitar convite continua escondido para o admin.
- Botão de ativar família só aparece quando o admin ainda não tem a família ativada.
- Convites pendentes agora têm ações diretas: copiar código, copiar link e cancelar.
- Usuários autorizados aparecem em uma lista própria.
- Espaçamentos revisados para iPhone.
