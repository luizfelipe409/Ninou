# Ninou v82.1.2 — Admin responsivo e limpo

- O admin web agora se adapta ao notebook e desktop com largura de até 1180 px, seguindo o comportamento responsivo da página inicial.
- As cinco abas administrativas formam uma grade estável em telas largas; no mobile continuam em uma faixa horizontal sem quebra de texto.
- Métricas passam de duas para quatro colunas no desktop, e listas de famílias, usuários e suporte usam duas colunas.
- A tela integral da família preserva a transição mobile, mas organiza os painéis em duas colunas no desktop.
- Todo painel administrativo legado permanece invisível durante carregamento, erro ou troca de runtime, eliminando checklists numerados, diagnósticos e cards antigos.
- Alertas de integridade repetidos são consolidados antes da renderização.
- “Executar checklist” foi substituído por “Verificar integridade” no mobile e no web app.
