# Ninou v75.25 — UX admin profissional

A v75.25 melhora a página de administração antes do fechamento da versão:

- Famílias aparecem como cartões humanos, com nome da família, contagem de membros e status.
- A rotina continua fechada por padrão e só abre por ação manual do admin.
- “Usuários autorizados” foi renomeado para “Membros desta família”.
- Nova área “Usuários encontrados no banco” lista contas localizadas no Firestore.
- Cada usuário encontrado pode ser convidado, autorizado como cuidador ou usado como origem de migração.
- Espaçamentos, botões, textos e cards do painel foram revisados para mobile.
