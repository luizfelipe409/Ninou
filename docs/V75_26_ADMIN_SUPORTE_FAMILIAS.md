# Ninou v75.26 — Admin como suporte e famílias/clientes isolados

- O admin do app não é tratado como membro da família Francisco.
- O bloco de membros mostra apenas pessoas realmente vinculadas à família selecionada.
- O admin aparece separadamente em “Acesso administrativo”.
- Convites são sempre vinculados à família/cliente selecionado no painel.
- O admin pode selecionar e abrir qualquer família cadastrada para suporte.
- A rotina familiar continua sem abrir automaticamente no painel admin.
- A migração usa a família selecionada como destino.
- Usuários encontrados no banco podem ser convidados, autorizados como cuidador ou migrados para a família selecionada.
